面试小知识点

发表于 2016-10-13   |   分类于 Java   |     |   阅读次数

TCP和UDP的区别：

  1. TCP传输控制协议，UDP用户数据包协议，都是传输层协议。
  2. TCP面向连接，UDP无连接
  3. TCP传输可靠，UDP传输不可靠，可能会丢包
  4. TCP传输速度慢，UDP传输速度快
  5. TCP保证数据正确性和数据的顺序，UDP可能丢包，不保证顺序
  6. UDP数据包结构简单，对系统资源要求少
