如何优雅地通过校招笔试

发表于 2017-10-18   |   分类于 随笔   |     |   阅读次数

我不是教你笔试如何作弊，而是教你笔试如何优雅地合作！

1. 使用谷歌浏览器和印象笔记

  • 首先，自行安装好谷歌浏览器和印象笔记，你不知道从哪里下载？好吧，看下图是如何下载印象笔记的。
  • 安装完印象笔记之后注册并登录。（印象笔记挺好用的，平时记笔记也很方便。）
  • 印象笔记里面建立群组共享笔记本，并把合作者加入到笔记本中。（创建共享笔记本的人需要是会员，具体操作暂不详述）

2. 在谷歌浏览器上安装印象笔记剪藏插件

  • 下载地址：印象笔记剪藏插件
  • 打开谷歌浏览器的扩展程序页面，如下图：
  • 安装插件，将下载的插件拖到该页面。如下图：

3. 登录印象笔记剪藏插件

安装完成后，谷歌浏览器的地址栏右边会出现插件的图标，第一次点击图标会提示登录，用你的印象笔记账号登录。见下图：

4. 将笔试的页面保存到印象笔记中

  • 假设你正在笔试，现在浏览器中点击下印象笔记剪藏的插件图标，会弹出保存按钮。
  • 你可以选择保存的内容，一般选整个页面（选择一次就好了，后面就默认这个）。也可以选择保存到的笔记本，如已经建好的校招笔试
  • （点击最先面的选项，可以进入插件的设置页面，可以设置总是保存到某个笔记本，这样以后保存内容的时候就默认的不需要选，只要点保存按钮就好了）

5. 其他小伙伴开始助攻

  • 其他小伙伴使用印象笔记电脑版，当然他也在共享的笔记本下，点击同步，就可以在笔记本中看到你保存的页面内容了。
