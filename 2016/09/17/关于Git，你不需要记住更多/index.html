为git命令设置别名

将以下内容追加到C:\Users\Troy\.gitconfig文件，就可以用简化命令，如git st来代替git status。

[alias]
co = checkout
ci = commit
br = branch
st = status
unstage = reset HEAD
last = log -1
lg = log –color –graph –pretty=format:’%Cred%h%Creset -%C(yellow)%d%Creset %s %Cgreen(%cr) %C(bold blue)<%an>%Creset’ –abbrev-commit
