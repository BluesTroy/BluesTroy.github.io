JDK1.8-1.5新特性总结

发表于 2016-10-09   |   分类于 Java   |     |   阅读次数

JDK1.8 新特性

  1. 接口的默认方法
    一个非抽象的方法实现，只需要使用 default关键字即可，这个特征又叫做扩展方法

  2. lambda表达式
    由参数列表、箭头符号->和函数体组成

  3. 函数式接口
    “函数式接口”是指仅仅只包含一个抽象方法的接口。只需要给你的接口添加 @FunctionalInterface 注解。

  4. 方法和对象函数引用
    Java8 允许使用::关键字来传递方法或者构造函数的引用。

  5. lambda直接访问外城的局部变量，无需声明final。

  6. lambda可访问对象字段和静态变量

  7. 包java.time下包含了全新的时间日期API，与Joda-time相似而不同。

JDK1.7 新特性

  1. switch中可以使用字符串

  2. 泛型实例化自动推断
    List tempList = new ArrayList<>();

  3. 语法上支持集合，并一定是数组
    List digits =[1,2,3,4];

  4. 新增取环境信息的方法
    File System.getJavaIoTempDir()//IO临时文件夹
    File System.getUserHomeDir()//当前用户目录
    …

  5. Boolean类型反转，空指针安全，参与位运算

  6. 两个Char间的equals
    boolean Character.equalsIgnoreCase(char c1,char c2);

  7. 安全的加减乘除
    int Math.safeAdd(int v1,int v2);

  8. 二进制字面量

  9. 数字字面量可出现下划线
  10. 一个catch多个异常，用“|”隔开
  11. 自定义自动关闭类
    实现AutoCloseable接口
  12. try-witch-resources，可自动关闭资源

JDK1.6 新特性

  1. Desktop类和SystemTray类
  2. STAX
  3. 轻量级Http Server API
  4. 用Console开发控制程序
  5. 对脚本语言支持
    …

JDK1.5 新特性

  1. 泛型
  2. foreach
  3. 自动拆箱装箱
  4. 枚举
  5. 静态导入（Static import）
  6. 元数据（Metadata）
  7. 线程池
  8. Java Generics
    创建一个明确只能存放某种特定类型对象的容器类实例。容器类型安全
